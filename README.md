# BD-TPI
Reportes: 
✅ 1. Listado de todos los usuarios registrados
Qué muestra: Nombre, ID, fecha de registro.

Utilidad: Permite ver quiénes están usando el sistema.

✅ 2. Listado de todos los mensajes enviados
Qué muestra: Usuario que lo envió, contenido del mensaje, fecha y hora.

Utilidad: Verifica el historial completo del sistema.

✅ 3. Cantidad total de mensajes por usuario
Qué muestra: Cada usuario con el número total de mensajes enviados.

Utilidad: Saber quién es más activo.

✅ 4. Cantidad total de mensajes por chat
Qué muestra: Cada chat (grupo o privado) con la cantidad de mensajes enviados.

Utilidad: Indica qué chats están más activos.

✅ 5. Mensajes enviados por día/hora (histograma de actividad)
Descripción: Agrupa los mensajes por fecha y hora para detectar patrones de uso.

Utilidad: Sirve para analizar los picos de tráfico y mejorar el rendimiento del sistema.
